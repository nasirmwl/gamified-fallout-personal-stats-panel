# Character stat panel

This repo is **boilerplate / a starting point** for a small static “stats” UI and daily checklist. The copy, percentages, labels, and placeholder image are **examples only** — they **do not describe or reflect any real person**, including the author(me).

Use it as a template: swap text, numbers, `avatar.svg`, and checklist items to match what you actually track.

### Features planned to be included
- char percentages connection to supabase logs to reflect more realistic percentages
- percentage calculations based on logs from supabase
- styling refactor 
- integration to nmwldb personalized [dashboard](https://github.com/nasirmwl/personal-dashboard-web)
